Editor.js

Mai 2020 --- DUT MMI

Lors d'un projet dans le cadre de mon DUT j'ai dû mettre en place un plugin JavaScript au sein d'une page web. Mon plugin était Editor.js.

Editor est un plugin éditeur de texte riche. Pour ce projet, je devais le mettre en place et expliquer comment j'avais fait. Pour expliquer, j'ai décidé de le faire directement au sein de l'éditeur pour rendre mon explication plus original.

L'utilisation de ce plugin m'a permis de comprendre la réelle importance de JavaScript ainsi que l'interaction qu'il permet de créer.

Créa'team

Juin 2020 --- DUT MMI

Dans le cadre de mon DUT, par groupe nous avons dû réaliser le site d'une agence de communication que nous devions créer. Nous avions créé Créa'team.

Faisant partie du groupe technique, j'avais pour objectif avec une camarade de réaliser le site de notre agence en utilisant le CMS Wordpress.

Lors de ce projet, nous avions dû faire de nombreuses recherches pour arriver à maitrîsé Wordpress comme c'était la première fois que nous l'utilisions. Avec l'aide de notre professeur nous avons réussi a réaliser un site design et moderne. Wordpress nous a permis de nous familiariser avec son extension Elementor qui est l'un des plus célèbres constructeurs de page Wordpress.

Enfin, nous avons également pu ajouter de la css au sein de notre site wordpress pour pouvoir le personnaliser au mieux.
